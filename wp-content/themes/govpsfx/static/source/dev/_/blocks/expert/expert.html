СТРАТЕГИЯ ТОРГОВЛИ:

Мультивалютный торговый форекс-робот Hyperboloid Multi представляет собой сеточного советника, который ищет точки входа в рынок и при поступлении сигнала от встроенных в него индикаторов, открывает рыночные ордера Buy и Sell в зависимости от поступившего сигнала.

Далее, если цена пошла в сторону открытого ордера, форекс-робот выставит тейк профит, указанный в настройках и при достижении цены уровня тейк профит, ордер закроется с прибылью. Если же цена пошла в противоположном направлении, то при прохождении ценой количества пунктов, указанных в настройках в параметре Distance, и при подтверждении сигналов от индикаторов, откроет ордер в том же направлении, но увеличенным лотом, на величину параметра Lot Multiplier, и переставит уровень тейк профита для обоих ордеров на величину указанной в настройках от цены открытия последнего ордера. Таким образом forex-советник Hyperboloid Multi усредняет открытые ордера, но уровень тейк профита находится ближе к текущей цене, чем при обычном усреднении. Если уже есть сетка ордеров одного направления и поступает сигнал на открытие ордера в другом направлении, то форекс-советник сделает это и дальше будет сопровождать новые ордера по той же стратегии. Таким образом, торговый робот работает сразу в обоих направлениях, что позволяет уменьшать просадку и поддерживать депозит в профите, при длительном движении цены в одну сторону.

Эту стратегию торговли, форекс-советник будет поддерживать и при выборе в настройках режима Recovery. В этом режиме, на графике, будут созданы дополнительные кнопки для открытия первого ордера на Buy или Sell. Далее торговый эксперт будет сопровождать эти ордера, вплоть до их полного закрытия.

ВАЖНО! Первый ордер, в режиме Recovery, должен быть открыт только при помощи этих кнопок, иначе торговый-робот не будет сопровождать ордера, открытые средствами торгового терминала. При выборе режима Recovery, для работы forex-советника необходимо в настройках оставить только одну рабочую валютную пару, и советник должен быть установлен только на график с выбранной валютной парой.

Также, для удобства торговли, на графике имеются три кнопки CloseAll, CloseBuy и CloseSell, при нажатии на которые будут закрыты соответствующие ордера только для ордеров, открытых на текущем графике.

В торговом форекс-роботе Hyperboloid Multi предусмотрена возможность использовать не все средства для торговли (параметр Reserve, in %), а часть средств зарезервировать.

Если использовать мультивалютную торговлю, на одном торговом счете, сразу по шести валютным парам, просадка будет небольшая, за счет того, что плюсовые сделки всегда будут перекрывать минусовые, а также, при открытии ордеров в разных направлениях, размер маржи, также уменьшается. При тестировании торгового форекс-робота в первой половине 2018 года, на демо и реальных счетах, используя начальный депозит в 2000 единиц, размер просадки не превышал 3% – 9%.

Советник выводит на график информацию о количестве открытых позиций и общему количеству открытых лотов отдельно для каждого направления торговли. Отображает информацию о профите, просадке и заработке за сегодня, вчера, за неделю и за текущий месяц, но только для той валютной пары, на графике которой установлен советник. Чтобы посмотреть эту информацию по другой валютной паре, просто перетащите мышью на текущий график эту валютную пару из окна «Обзор рынка».

MONEY MANAGEMENT

  • Reserve in % = 0.0; — процент резервных средств, которые не будут использоваться в торговле.
  • Use Autolot = true; — true — использовать автолот; false — не использовать.
  • Percent For Autolot = 1; (Для InstaForex умножить на 10) — процент средств для расчета автолота.
  • Fix Lot = 0.01; (Для InstaForex умножить на 10) — фиксированный лот для торговли. При автолот = true не используется.
  • Take Profit, points = 57; — тейк профит.
  • Distance, points = 43; — дистанция для сетки ордеров.
  • Lot Multiplier = 2; — на сколько умножать следующий лот в сетке ордеров.

CONFIGURE INDICATORS

  • Indicator Period = 19; — период первого индикатора.
  • Indicator Shift = 1; — отклонение индикатора в барах.
  • Indicator Method = Simple; — метод усреднения индикатора.
  • Indicator Applied Price = Close price; — к какому уровню цены применять индикатор.
  • Delta Step, 0.01-0.07 = 0.02; — шаг изменения цены для второго индикатора. Диапазон значений от 0.01 до 0.07.
  • Delta Maximum, 0.1-0.5 = 0.2; — максимальный шаг для второго индикатора. Диапазон значений от 0.1 до 0.5.

SETTINGS UNIT INFORMATION

  • Show Info Table = TRUE; — показывать или нет блок информации на графике
  • Font Size = 10; — размер шрифта для блока информации и кнопок.
  • Magic Number = 180318; — магический номер для отличия своих ордеров от других.
